Cara kerja

Tiga playground,
satu platform.

PLAYGROUND 01

Hit Saurce

Hit manapun. Lihat response, headers, timing dalam satu tampilan bersih tanpa konfigurasi.

PLAYGROUND 02

Runner

Susun API jadi skenario E2E. Token dari login otomatis tersedia di request berikutnya tanpa tulis script.

PLAYGROUND 03

Performance

Load test langsung dari browser. Pilih preset jumlah user atau atur sendiri, hasilnya real-time.

HASIL

Report siap bagikan

Satu klik untuk generate laporan. Bagikan link ke tim atau klien tanpa perlu export file.

Fitur

Semua yang kamu butuhkan.
Tidak lebih, tidak kurang.

Setiap fitur dipilih berdasarkan apa yang benar-benar dipakai testing.

Request Builder

Visual, bukan kode

Isi method, URL, headers, dan body lewat form intuitif. Tidak perlu hafal format cURL atau syntax JSON.

POST /v1/auth/login
Authorization: Bearer {{token}}
Content-Type: application/json
Variable Extraction

Ambil data antar request

Simpan nilai dari response seperti token atau ID lalu gunakan di request berikutnya otomatis tanpa script.

Login - simpan {{token}} dari data.token
Get Cart - pakai {{token}} di header
Checkout - pakai {{cart_id}}
Assertions

Validasi otomatis, hasil jelas

Tentukan ekspektasi untuk status code, isi response, dan waktu. Setelah send langsung tahu pass atau fail.

OK Status code = 200
OK Response time < 500ms
FAIL data.user.email exists
Environment

Dev, Staging, Prod - 1 klik

Simpan base URL dan variabel per environment. Switch antara Dev, Staging, dan Production tanpa edit manual.

Dev    : api.dev.myapp.id
Staging : api.stg.myapp.id
Prod   : api.myapp.id

FAQ

Pertanyaan yang sering ditanya.

Apakah tier gratis benar-benar gratis selamanya? +
Ya. Tidak ada batas waktu, tidak perlu bayar dulu. Upgrade hanya jika kapasitas kamu bertumbuh.
Apakah perlu install sesuatu? +
Tidak sama sekali. HITSaurce berjalan sepenuhnya di browser tanpa ekstensi atau desktop app.
Bagaimana cara kerja load test dengan K6? +
Kamu mengatur jumlah virtual user dan durasi lewat UI. HITSaurce menjalankan K6 di server kami dan streaming hasilnya secara real-time ke browser tanpa install K6 lokal.
Bisa import dari Postman? +
Ya, import collection Postman format JSON tersedia di semua tier termasuk gratis.
Apakah data request saya aman? +
Semua komunikasi dienkripsi via HTTPS. Data kamu tidak pernah dijual atau dibagikan ke pihak lain.
