Fitur

Semua yang kamu butuhkan.
Tidak lebih, tidak kurang.

Setiap fitur dipilih berdasarkan apa yang benar-benar dipakai testing sehari-hari — bukan fitur demi fitur.

Request Builder

Visual, bukan kode

Isi method, URL, headers, dan body lewat form yang intuitif. Tidak perlu hafal format cURL atau syntax JSON.

POST /v1/auth/login
Authorization: Bearer {{token}}
Content-Type: application/json
Variable Extraction

Ambil data antar request

Simpan nilai dari response (seperti token atau ID) lalu gunakan di request berikutnya — otomatis tanpa script.

Login → simpan {{token}} ← data.token
Get Cart → pakai {{token}} di header
Checkout → pakai {{cart_id}}
Assertions

Validasi otomatis, hasil jelas

Tentukan ekspektasi — status code, isi response, waktu. Setelah send, langsung tahu apa yang pass dan fail.

✓ Status code = 200
✓ Response time < 500ms
✗ data.user.email = "test@…"
Environment

Dev, Staging, Prod — 1 klik

Simpan base URL dan variabel per environment. Switch antara Dev/Staging/Production tanpa edit apapun secara manual.

🟢 Development → api.dev.myapp.id
🟡 Staging    → api.stg.myapp.id
🔴 Production → api.myapp.id

FAQ

Pertanyaan yang sering ditanya.

Apakah tier gratis benar-benar gratis selamanya?+
Ya. Tidak ada batas waktu, tidak perlu bayar. Upgrade hanya jika kapasitas kamu bertumbuh.
Apakah perlu install sesuatu?+
Tidak sama sekali. HitSaurce berjalan sepenuhnya di browser — tidak ada ekstensi, tidak ada desktop app.
Bagaimana cara kerja load test dengan K6?+
Kamu mengatur jumlah virtual user dan durasi lewat UI. HitSaurce menjalankan K6 di server kami dan streaming hasilnya secara real-time ke browser — tanpa install K6 lokal.
Bisa import dari Postman?+
Ya, import collection Postman format JSON tersedia di semua tier termasuk gratis.
Apakah data request saya aman?+
Semua komunikasi dienkripsi via HTTPS. Data kamu tidak pernah dijual atau dibagikan ke pihak lain.
